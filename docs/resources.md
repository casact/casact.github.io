# Resources

This page contains resources intended to guide the practitioner from writing basic scripts to maintaining
production-ready software.

## Programming Languages

- [The Python Tutorial](https://docs.python.org/3/tutorial/) - the official, beginner-friendly walkthrough of the language
- [Automate the Boring Stuff with Python](https://automatetheboringstuff.com/) - a free, practical introduction aimed at non-programmers
- [Real Python](https://realpython.com/) - tutorials ranging from basics to advanced topics
- [R for Data Science](https://r4ds.hadley.nz/) - the standard modern introduction to R
- [An Introduction to R](https://cran.r-project.org/doc/manuals/r-release/R-intro.html) - CRAN's own reference manual

## Git & GitHub

- [Git Handbook](https://docs.github.com/en/get-started/using-git/about-git) - GitHub's introduction to how Git works
- [Pro Git](https://git-scm.com/book/en/v2) - the free, comprehensive Git book

## Documentation

- [Sphinx](https://www.sphinx-doc.org/) - the documentation generator used to build this site
- [Read the Docs](https://docs.readthedocs.io/) - free hosting for project documentation
- [roxygen2](https://roxygen2.r-lib.org/) - in-source documentation for R packages

## Software engineering practices

- [Semantic Versioning](https://semver.org/) - a standard convention for versioning released software
- [pytest documentation](https://docs.pytest.org/) - the standard Python testing framework
- [testthat](https://testthat.r-lib.org/) - the standard R testing framework
- [Python Packaging User Guide](https://packaging.python.org/) - how to structure, build, and publish a Python package
- [R Packages (2nd edition)](https://r-pkgs.org/) - Hadley Wickham and Jenny Bryan's guide to building R packages
- [GitHub Actions documentation](https://docs.github.com/en/actions) - CI/CD directly from a GitHub repository
- [PEP 8](https://peps.python.org/pep-0008/) - the standard Python style guide
